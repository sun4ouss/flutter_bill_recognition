# Голосовий вивід у Flutter додатку

## 🎯 Реалізація

### 1. Додаткові залежності
Додати в `pubspec.yaml`:
```yaml
dependencies:
  flutter_tts: ^3.8.3  # Text-to-Speech
  permission_handler: ^11.0.1
```

### 2. Голосовий сервіс (`voice_service.dart`)
- **Flutter TTS** - синтез мовлення
- **Українська мова** - "uk-UA"
- **Розумні повідомлення** - залежно від точності

### 3. Інтеграція в головний додаток
- **Автоматичний вивід** після розпізнавання
- **Кнопка повтору** - "Повторити голосом"
- **Кнопка зупинки** - "Зупинити"

## 📱 Функціонал

### Повідомлення залежно від точності:
- **≥85%**: "Номінал $denomination доларів США з точністю X%"
- **70-84%**: "Ймовірно, це $denomination доларів США. Точність X%"
- **<70%**: "Банкноту не розпізнано. Спробуйте ще раз."

### Голосові параметри:
- **Мова**: Українська (uk-UA)
- **Швидкість**: 0.8 (нормальна)
- **Гучність**: 1.0 (максимум)
- **Тон**: 1.0 (стандартний)

## 🔧 Встановлення

```bash
# Встановити залежності
flutter pub get

# Запустити додаток з голосом
flutter run --dart-define=USE_VOICE=true
```

## 📋 Дозволи

Для Android додати в `android/app/src/main/AndroidManifest.xml`:
```xml
<uses-permission android:name="android.permission.MODIFY_AUDIO_SETTINGS" />
```

## 🎯 Використання

1. **Автоматичний вивід** - після розпізнавання
